Rubik's Cube SVG Generator - Interactive Tester

Input

Format: 54 characters representing all stickers
U-face (white) = 0-8
D-face (yellow) = 9-17
F-face (red) = 18-26
B-face (orange) = 27-35
R-face (blue) = 36-44
L-face (green) = 45-53
Examples: R U' R2 R U R' U'
Space-separated moves are processed in order.

How to Use

Option 1: Browser-based (requires Python server)
1. Run: python test_server.py
2. Enter state and moves above
3. Click "Generate SVG"

Option 2: Command line (always works)
python scripts/gen_rubik_guide.py --state "www..." --moves "R U R'" --output result.svg
Then open result.svg in a browser.
